Як працює SupportBot

AI-бот для Signal-груп технічної підтримки

Що це

SupportBot моніторить вашу Signal-групу, автоматично знаходить вирішені проблеми, зберігає їх у базу знань та відповідає на нові запитання, використовуючи попередній досвід.

Як додати до групи

  1. Зверніться до адміністратора за номером бота
  2. В Signal: налаштування групи → Додати учасників → номер бота
  3. Адмін групи підтверджує реєстрацію
  4. Бот починає працювати автоматично

Як користуватися

  • Бот сам вирішує, коли відповідати — якщо знає відповідь
  • Згадайте бота напряму для примусового виклику
  • Бот розуміє текст, зображення та скріншоти
  • Не відповідає на привітання, «ок», емодзі
Конфіденційність: бот обробляє повідомлення для створення бази знань. Використовуйте лише зі згоди учасників групи.

Імпорт історії

Якщо в групі вже є вирішені проблеми:

  1. Адмін запитує токен через POST /history/token
  2. Сканує QR-код через GET /history/qr/{token} у Signal
  3. Бот автоматично витягне кейси з історії

Технічні деталі

  • LLM: Google Gemini
  • Embeddings: text-embedding-004
  • Vector DB: Chroma
  • Database: MySQL 8.0
  • Signal: signal-cli (неофіційний)

How SupportBot Works

AI bot for Signal technical support groups

What it is

SupportBot monitors your Signal group, automatically extracts solved issues into a knowledge base, and answers new questions using past experience.

Adding to a group

  1. Contact your admin for the bot's phone number
  2. In Signal: group settings → Add members → bot number
  3. Group admin confirms registration
  4. Bot starts working automatically

How to use

  • Bot decides on its own when to reply — only if it has an answer
  • Mention the bot directly for a forced response
  • Understands text, images, and screenshots
  • Ignores greetings, "ok", emoji-only messages
Privacy: the bot processes messages to build a knowledge base. Only use with the consent of group members.

History import

If the group already has solved issues:

  1. Admin requests a token via POST /history/token
  2. Scans QR code from GET /history/qr/{token} in Signal
  3. Bot automatically extracts cases from history

Technical details

  • LLM: Google Gemini
  • Embeddings: text-embedding-004
  • Vector DB: Chroma
  • Database: MySQL 8.0
  • Signal: signal-cli (unofficial)
